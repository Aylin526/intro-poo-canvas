# 🎨 Objetos 2D con Canvas y Programación Orientada a Objetos

## 📖 Descripción del Proyecto

Este proyecto demuestra el uso del elemento **Canvas de HTML5** junto con **JavaScript** y el paradigma de **Programación Orientada a Objetos (OOP)** para crear y renderizar objetos gráficos en pantalla.

La aplicación genera círculos dentro de diferentes áreas del canvas, aplicando propiedades como posición, radio, colores y texto. Además, se utilizan valores aleatorios para mostrar cómo los objetos pueden generarse dinámicamente dentro de los límites definidos del lienzo.

El objetivo principal es comprender cómo estructurar objetos gráficos mediante clases en JavaScript y cómo representarlos visualmente utilizando la API de Canvas 2D.

---

## 🎯 Objetivo

Implementar una aplicación web que permita visualizar objetos gráficos en un **Canvas 2D**, utilizando **clases en JavaScript** para representar figuras geométricas y aplicar conceptos de **Programación Orientada a Objetos** como encapsulación y reutilización de código.

---

## 🧩 Funcionalidades de la Aplicación

### 1️⃣ Objeto 2D

Se genera un círculo centrado en el canvas utilizando valores definidos manualmente para demostrar el uso básico de la clase.

### 2️⃣ Objeto con Posición Aleatoria

Se genera un círculo con posición y tamaño aleatorio, asegurando que el objeto permanezca dentro de los límites del canvas.

### 3️⃣ Múltiples Objetos

Se crean múltiples círculos utilizando un arreglo de objetos, cada uno con colores generados aleatoriamente y renderizados en el canvas.
